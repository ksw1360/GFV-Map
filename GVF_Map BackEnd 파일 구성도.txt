SpringBoot 4.0.3
Dependency용도
                spring-boot-starter-data-jpa : JPA + Hibernate — DB 접근 (Repository)
                spring-boot-starter-security : Spring Security — 인증/인가
                spring-boot-starter-validation : Bean Validation — @Valid, @NotNull, @Email 등
                spring-boot-starter-webmvc : Spring MVC — REST API (Controller, RequestMapping)
                lombok (compileOnly + annotationProcessor) : Lombok — @Getter, @Builder 등
                spring-boot-devtools : DevTools — 코드 변경 시 자동 재시작
                mysql-connector-j : MySQL 드라이버
